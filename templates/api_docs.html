Portfolio API

REST API za moje projekte. Koristi se za fetch podataka.

API Endpoints
Metoda URL Opis
GET /api/projects/ Lista svih projekata
GET /api/projects/?featured=true Samo istaknuti projekti
GET /api/projects/?search=django Pretraga projekata
GET /api/projects/<id>/ Detalji određenog projekta
Primer API poziva

// JavaScript fetch primer
fetch('/api/projects/')
  .then(response => response.json())
  .then(data => console.log(data));

// Python requests primer
import requests
response = requests.get('http://tvoj-sajt.com/api/projects/')
projects = response.json()
Tehnologije API-ja
  • Django REST Framework
  • Filter, Search, Sort
  • Read-only API
API je read-only. Ne zahteva autentifikaciju za GET zahteve.
